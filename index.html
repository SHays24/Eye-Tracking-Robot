About the Design

Communicating the engineering intent behind the system.

The Creepy Eyes Tracking System is an interactive device designed for use in Halloween props, museum exhibits, or playful public installations. The system detects a person’s position using ultrasonic sensors and rotates a pair of eyes to “look at” the target in real time.

The design emphasises:

  • Stable and smooth tracking using a software smoothing function.
  • Robust ultrasonic distance readings with left/right comparison logic.
  • Clear visual feedback with RGB LED colour states.
  • A compact internal 3D structure that securely mounts all components.

How It Works

From ultrasonic distance readings to expressive servo motion.

Control Logic

An Arduino reads distance data from two ultrasonic sensors. A custom function facingDir() compares left and right readings and chooses one of four states:

  • Centre (0): distances are similar → eyes look straight.
  • Left (1): object closer on left → eyes rotate left.
  • Right (2): object closer on right → eyes rotate right.
  • No Target (3): both sensors out of range → LEDs turn green.

Smoothing & Motion

Instead of jumping directly to the new angle, the system uses: 

smoothIt(&servoSmoothedValue, value);
servoL.write(servoSmoothedValue);
servoR.write(servoSmoothedValue);

This exponential smoothing (10% of the difference per loop) creates natural, eye-like motion rather than mechanical snapping.

Hardware Overview

  • Microcontroller: Arduino (project prototype board).
  • Two ultrasonic sensors for left/right distance measurement.
  • Two servos to control the eye direction simultaneously.
  • RGB LED strip (Neopixel) for system state indication.
  • Custom 3D internal frame for mounting and cable routing.

System Engineering Techniques

  • Modular testing of sensors, servos, and LEDs before integration.
  • Incremental integration to isolate and fix issues early.
  • Real-world scenario testing with human movement paths.
  • FMEA updated weekly using test outcomes from Week-6.
